Getting Started

Download or install from one of these sources

npm install basscss
bower install basscss
git clone https://github.com/basscss/basscss.git
Download Source

Or use the CDN link:

<link href="http://d2v52k3cl9vedd.cloudfront.net/basscss/6.0.0-beta.2/basscss.min.css" rel="stylesheet">
